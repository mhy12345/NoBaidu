== Description ==
 
This plugin are designed for chinese wordpress user's who do not like Baidu and want to block all visitors came from Baidu. Also, you can use this plugin to change robots.txt of your site to block baidu-spider.
 
== Installation ==
 
1. Upload the whole directory of `no-baidu` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
